🤖 My Business WhatsApp Bot
⚡ Quick Setup (5 minutes)
Step 1: Install Node.js
nodejs.org se download karein
Step 2: Files Copy Karein
```
my-bot/
  ├── bot.js
  └── package.json
```
Step 3: Install Karein
```bash
cd my-bot
npm install
```
Step 4: Apni Info Bharein
bot.js mein CONFIG section update karein:
ownerName: "Aapka Naam"
ownerPhone: "923001234567"
whatsapp: "0300-1234567"
email: "aap@email.com"
Step 5: Bot Chalayein
```bash
node bot.js
```
Step 6: QR Scan Karein
WhatsApp > Linked Devices > Link a Device > QR Scan
